# Repositorio de Recursos para el PD (Diploma)

Este repositorio contiene el código para una página web sencilla que recopila recursos digitales para estudiantes del Programa del Diploma (PD) del Bachillerato Internacional (IB).

## Objetivo

El objetivo de esta página es ofrecer un punto de acceso a bases de datos académicas, gestores de bibliografía y herramientas de apoyo para los componentes troncales (Monografía, TdC, CAS) y la investigación avanzada.

## Vista Previa

Puedes ver la página en funcionamiento aquí: `https://hlanchasteacher.github.io/DPResources/index.html`
